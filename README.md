# 10-Week-Programming-Exercises-for-Beginners
Several common question types on UVA online judge

**初學者 10 週程式設計練習清單**

🎯 適用對象

* 大一資訊系 / 非資工但需學程式的學生
* 具備 **C / C++ / Java / Python 任一語言基礎語法**
* 尚未熟悉演算法與題目拆解

【課程能力指標代碼說明】

* C1：基本程式語法與流程控制
* C2：問題拆解與演算法思維
* C3：陣列與字串處理能力
* C4：資料結構應用（stack / queue / heap）
* C5：數學與邏輯建模能力
* C6：遞迴與回溯思維
* C7：效能與時間複雜度意識
* C8：完整程式實作與除錯能力

**📌 教學 / 自學建議（很重要）**

* **每週至少 AC 2 題**
* 每題寫完後請務必完成：
  1. ✍️ 列出「我卡在哪」
  2. ✍️ 列出「我學到什麼」
* ❌ 不要一開始就看解答
* ✅ 先用「暴力解」寫出來，再優化
* ✅ 養成自己產生測資的習慣
